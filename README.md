hasil modifikasi dari https://github.com/Datalux/Osintgram.git
jadi bisa dibuka mudah di web tanpa harus tulis perintah di terminal

cara pakai: clone dulu atau downlaod, ambil token dari hikerapi.com masuk atau daftar, copas API nya, buka config/credentials.ini tambahkan token hikerapi_token, save
buka folder proyek yang sudah di clon atau download, buka file run.bat ctrl+klik http://127.0.0.1:5000 atau buka browser cari http://127.0.0.1:5000
done!!

cara akhiri program: tutup tab http://127.0.0.1:5000, buka terminal yang berjalan klik ctrl+c tekan Y dan enter
